# Execution

This directory is the future home for in-repo execution guidance used by operators and automation workers.

It is intended for launch prompts, ticket workflows, runbooks, clean commit guidance, reusable execution templates, and apply-to-project material. Content here should help someone carry out work using AEGIS conventions without redefining the canonical roles themselves.

Canonical role behavior remains in `skills/` and `contracts/`. The `execution/` area may reference those sources, but it should not replace them or become the source of truth for skill behavior, contracts, or provider documentation.

The existing `templates/aegis-execution/` material is intentionally not migrated, copied, moved, or deleted as part of this skeleton. Later tickets can decide what belongs here and how it should be adapted.

## Areas

- `prompts/`: operator and launch prompt material to be added by later tickets.
- `runbooks/`: repeatable execution procedures and ticket workflows to be added by later tickets.
- `templates/`: reusable execution templates and apply-to-project material to be added by later tickets.
