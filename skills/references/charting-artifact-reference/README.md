# Charting Artifact Reference

## Scope

Reference scope: Axis-2 knowledge for chart artifacts: the shared facts,
constraints, and quality expectations that future chart procedures should use
when producing chart deliverables.

It is not a procedural skill and is not independently triggerable.

## Consuming Skills

- `chart-artifact-generation`

## Sections

| id | topic | open when |
| --- | --- | --- |
| claim-and-chart-type | Analytical claim and chart type fit. | Open when a chart artifact needs its intended analytical comparison or chart form checked. |
| context-and-encoding | Titles, labels, legends, units, source notes, and visual encodings. | Open when a chart artifact needs interpretability or encoding guidance. |
| scales-and-styling | Scale choices, transformations, styling, contrast, and readability. | Open when scale, styling, accessibility, or readability choices materially affect interpretation. |
| export-legibility | Export legibility, intended size, format, metadata, and reuse. | Open when a chart artifact is being exported, saved, reused, or inspected at final size. |

## Out Of Scope

- Procedural trigger logic, workflow steps, or activation rules.
- Role prompt content.
- Implementation details for any specific chart worker.
